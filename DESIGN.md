## 四、编辑器面板

### 4.1 编辑器头部

- 左侧：「Markdown 编辑」标题
- 右侧操作按钮：
  - **📋 模板** — 打开模板库弹窗

⌘F 触发显示，位于格式工具栏下方：

```
┌─────────────────────────────────────────┐

### 4.4 编辑器主体

- **行号区域**：左侧灰色行号（可在设置中关闭）
- **Textarea**：主编辑区，支持：

  - ⌘V 粘贴图片
  - 拖拽图片上传
  - ⌘Z 撤回 / ⌘⇧Z 重做（最多 50 步）

### 4.5 编辑器底栏

```

## 五、预览面板

### 5.1 预览头部操作栏

```
[预览 📱375px]         [历史] [保存] [导出▾] [复制到公众号]
```

| 按钮 | 功能 |
|------|------|
| **历史** | 打开历史记录侧栏 |
| **保存** | 保存到 localStorage（⌘S） |
| **导出 ▾** | 下拉菜单 |
| **复制到公众号** | 蓝色主按钮（⌘⇧C） |

### 5.2 导出下拉菜单

```
┌────────────────┐
│ 📄 导出 HTML   │
│ 📑 导出 PDF    │
│ 🖼️ 导出长图    │
│ 📝 导出 Word   │
│ ──────────────│
│ 🚀 分发到多平台 │ ← WechatSync SDK
└────────────────┘
```

- 导出文件名自动使用文章第一个 `#` 标题
- 导出长图截取完整文档（非仅可见区域）
- 分发到多平台：需安装 WechatSync Chrome 扩展

### 5.3 预览主体

- 对 `markdownText` 做 300ms 防抖渲染
- 应用当前主题的内联样式
- 代码块高亮（Highlight.js + Mac 风格红黄绿圆点）
- Mermaid 图表渲染
- 手机预览模式下限宽 375px，居中显示

## 七、图片处理系统

### 粘贴/拖拽图片流程

```
用户粘贴图片
    ↓
SM.MS 图床开启？
    ├── 是 → 上传到 SM.MS → 获取永久链接 → ![name](https://url)
    └── 否 → FileReader 转 Base64
               ↓
         生成唯一 ID (img_xxx)
         存入 imageStore (localStorage)
         编辑器显示: ![name](img:img_xxx)  ← 短占位符
               ↓
         渲染/导出时 → 展开为真实 Base64 数据
```

## 九、视觉规范

### 配色

```css
/* 亮色模式 */
--bg:          #ffffff
--bg-secondary: #f8f9fb
--text:         #1a1a2e
--text-muted:   #8896a6
--border:       #e8ecf1
--primary:      #4f6ef7  (蓝色主色)
--primary-hover: #3d5ce5

/* 暗色模式 */
--bg:           #1a1a2e
--bg-secondary:  #252540
--text:          #e8ecf1
--text-muted:    #8896a6
--border:        #2d2d4a
--primary:       #4f6ef7
```

### 字体

```css
--font-sans:  -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif
--font-serif: Georgia, 'Noto Serif SC', serif
--font-mono:  'SFMono-Regular', Consolas, 'Liberation Mono', monospace
```

### 圆角

```css
按钮: 8px
面板: 12px
输入框: 6px
卡片: 10px
```

### 交互动效

- 按钮悬停：`transition: all 0.2s ease`
- 面板进入/退出：`transition: transform 0.3s ease, opacity 0.2s`
- Toast 通知：底部弹出，2s 后自动消失
- 主题切换：预览区平滑过渡

## 十二、数据持久化

所有数据存储在 `localStorage`：

| Key | 内容 |
|-----|------|
| `md-converter-draft` | 编辑器草稿内容 |
| `md-converter-history` | 保存的历史记录列表 |
| `md-converter-image-store` | Base64 图片存储 Map |
| `md-converter-image-history` | 图床上传历史 |
| `md-converter-imagehost` | 图床开关状态 |
| `md-converter-deepseek-key` | DeepSeek API Key |
| `md-converter-word-goal` | 字数目标设置 |
| `md-converter-linenumbers` | 行号显示开关 |
| `md-converter-custom-css` | 自定义 CSS 代码 |
